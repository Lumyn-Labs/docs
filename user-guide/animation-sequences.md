# Animation Sequences

Create and edit animation sequences, then add them to a device configuration.

## Overview

The **Animation Sequences** tab lists your sequences. Create a new one with **Create sequence**.

## Creating a sequence

Creating a sequence opens the sequence editor. Use the **Settings** button (gear icon) in the toolbar to set:

- ID: Used to reference the sequence.
- Name: Human-readable title.
- Description and tags (optional).

Studio includes 26 built-in animations: Fill, Blink, Breathe, RainbowRoll, SineRoll, Chase, FadeIn, FadeOut, RainbowCycle, AlternateBreathe, GrowingBreathe, Comet, Sparkle, Fire, Scanner, TheaterChase, Twinkle, Meteor, Wave, Pulse, Larson, Ripple, Confetti, Lava, Plasma, and Heartbeat.

## Editing a Sequence

Use the editor to add, remove, and reorder steps.

```{image} ../assets/lumyn-studio/animation-sequences/animation-sequence-editor.png
:alt: Animation Sequence Editor
```

- **Toolbar**: Back to the list, sequence name, **Settings** (ID, Name, Description, Tags), **Add to Device**, and **Save**.
- **Left column — Animation Preview**: Simulated LED strip with zone/custom target selection, LED count, play/pause, and state scrubber.
- **Left column — Animations**: Search box, sort (Name, Speed, Complexity), and a grid of animation preview cards. Click a card to add or edit a step.
- **Right column — Sequence steps**: Play/stop for the whole sequence, a step navigation slider with previous/next, and an expandable list of steps. Each step shows the animation name, delay, iterations, reversed state, and color. Expand a step to edit delay (ms), iterations (1–20), reversed, and color. Use **Add Step**, **Duplicate**, **Move Up**, **Move Down**, and **Remove** to manage steps.
