# Creating Custom Modules

Use Lumyn Studioâ€™s Create Custom Module to define module metadata, payload, and firmware.

## Workflow

1. Basic Information: Type, name, category, tags, connection types, images, links.
2. Payload: Define up to 16 bytes of data returned each poll.
3. Custom Configuration: Module-specific options available in code.
4. Readme: Author markdown documentation for sharing/maintenance.
5. Generated Firmware: Generate and download starter firmware once inputs are complete.

See the [Modules](user-guide/modules) page for UI details and screenshots.
